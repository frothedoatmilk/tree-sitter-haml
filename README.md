# tree-sitter-haml

This is a general tree-sitter parser grammar for [the HAML templating language](https://haml.info/).

## Development

### Building

To build the grammar, run:

```shell
npm run build
```

### Testing

To test the grammar, run:

```shell
npm run test
```
